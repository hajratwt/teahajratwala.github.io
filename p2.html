Project 2: Ann Arbor GO

Video Demo

Responsive YouTube Video Embed

An AR project made with Unity AR tools. Like Pokemon Go, you walk around a map of your location doing stuff, occasionally fighting squirrels.

Unlike Pokemon Go, it's tree themed and you play as Barack Obama.

We wanted to add a feature we wished Pokemon GO had had: step counting (it tracks distance travelled, but not steps specifcally). You'll see in the video that we did exactly that. Obama recieves coins proportional to the number of steps taken multiplied by the number of (squirreless) trees spawned.
